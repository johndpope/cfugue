CFugue MusicNote Library

Directory Structure:

\include: contains the #include files 

\lib: contains the library files for static linking and dynamic linking

\bin: contains the dynamic linked dlls and sample test application executables

\src: contains the source code

\test: contains the test application source code

\build: contains the VC solution/project files to build the source code. Also contains the Doxyfile to generate the documentation

\docs: contains the Doxygen generated documentation

\guide: contains the CFugue Complete Programming Guide Source
